# Desafio Android - Curso em Vídeo
Projeto feito durante o curso de HTML5 e CSS3 do [Curso em Vídeo](http://cursoemvideo.com/)
## Objetivo
O foco principal deste projeto foi aplicar conceitos fundamentais de design web e garantir que o site seja acessível em diferentes tamanhos de tela.

## Tecnologias e Conceitos Aprendidos

### Responsividade
*   Criação de interfaces que se adaptam a diversos dispositivos (TV, celular, tablets e computadores).

### Estilização com CSS3
*   **Variáveis:** Uso de `:root` para facilitar a manutenção de cores.
*   **Cores:** Manipulação de `color` e `background-color`.
*   **Bordas e Sombras:** Aplicação de `border-radius`, `text-shadow` e `box-shadow` para profundidade.
*   **Interatividade:** Uso de pseudo-classes como `:hover`.

### Estrutura Semântica
*   Uso correto de tags como `<header>`, `<nav>`, `<main>`, `<article>` e `<footer>` para melhor SEO e acessibilidade.
